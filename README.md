# DAM Cloud API

API REST desenvolupada amb Spring Boot i desplegada al núvol amb Render.

## Endpoints

GET /api/books  
GET /api/books/{id}  
POST /api/books  
PUT /api/books/{id}  
DELETE /api/books/{id}  

## Producció

URL:

https://prova-cloud-computing.onrender.com/

## Execució local

Configura:

DB_URL  
DB_USER  
DB_PASSWORD  

Executa:

./mvnw spring-boot:run
